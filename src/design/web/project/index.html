Projecte

OpenPlenary és un projecte software creat per acostar i obrir els plens municipals i donar a coneixer als ciutadans i ciutadanes la tasca que fan els regidors el·lectes. També es pot fer servir en altres ambits com per exemple assemblees, associacions i en general qualsevol tipus d'organització on s'hagin de prendre decisions de manera col·lectiva.

Està desenvolupat amb lleguatge Groovy fent servir el framework Grails.

Instal·lació

Pots baixar-te el codi font de GitHub

Un cop el tinguis baixat, donat que és un projecte Grails, segueix les següents instruccions

Quan acabis ves desde linea de comandes al directory on tinguis el projecte i executa: 

> grails compile

I després: 

> grails run-app

Si arrenca l'aplicació ja pots començar a tocar el codi per adaptar-lo a les teves necessitats!
