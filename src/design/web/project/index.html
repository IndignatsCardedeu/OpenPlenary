Projecte

OpenPlenary és un projecte software desenvolupat pels Indignats i Indignades de Cardedeu per obrir els plens municipals i donar a coneixer als ciutadans i ciutadanes la tasca que fan els regidors el·lectes. També es pot fer servir en altres ambits com per exemple assemblees, associacions i en general qualsevol tipus d'organització on s'hagin de prendre decisions de manera col·lectiva.

Està desenvolupat amb lleguatge Groovy fent servir el framework Grails.

Instal·lació

Descarrega el codi font d'Open Plenary

Open Plenary fa servir las versió 2.1.1 del framework Grails (pot funcionar amb versions superiors pero no l'hem provat). Descarrega el framework Grails i segueix les següents instruccions.

Quan acabis ves a la linea de comandes i des del directory on tinguis el projecte executa: 

> grails compile

I després: 

> grails run-app

Si arrenca l'aplicació ja pots començar a tocar el codi per adaptar-lo a les teves necessitats!
