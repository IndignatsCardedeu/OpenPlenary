Projecte

OpenPlenary és un projecte software desenvolupat pels Indignats i Indignades de Cardedeu per obrir els plens municipals i donar a coneixer als ciutadans i ciutadanes la tasca que fan els regidors el·lectes. També es pot fer servir en altres ambits com per exemple assemblees, associacions i en general qualsevol tipus d'organització on s'hagin de prendre decisions de manera col·lectiva.

Està desenvolupat amb lleguatge Groovy fent servir el framework Grails.

Instal·lació

Per instal·lar OpenPlenary necessitaras un servidor d'aplicacions J2EE com Tomcat i, tot i que funciona per defecte amb H2, és recomanable fer servir una base de dades com MySQL o MariaDB.

Una vegada ho tinguis només cal seguir les instruccions següents:

  • Descarrega la última versió d'Open Plenary
  • Descomprimeix el fitxer zip
  • Desplega el fitxer war en el servidor d'aplicacions

Amb això tindràs funcionant l'aplicació amb la configuració per defecte i una base de dades efímera. Per customitzar el teu OpenPlenary, copia el fitxer openplenary-config.groovy.sample al path d'execució del servidor (en el cas de Tomcat seria a l'arrel del path on es troba instal·lat), renombra-ho com openplenary-config.groovy i edita-ho segons les teves necessitats. Ves a la secció de Configuració per a més informació.

Compilació

Si enlloc d'instalar-te el WAR prefereixes compilar tu mateix el projecte i remenar una mica, segueis les següents instruccions:

Descarrega el codi font d'Open Plenary

Open Plenary fa servir las versió 2.2.4 del framework Grails (pot funcionar amb versions superiors pero no l'hem provat). Descarrega el framework Grails i segueix les següents instruccions.

Quan acabis ves a la linea de comandes i des del directory on tinguis el projecte executa: 

> grails compile

I després: 

> grails run-app

Per administrar el contingut accedeix a http://localhost:8080/openplenary/admin (usuari admin, password admin)

Configuració

Pots configurar l'aplicació creant el fitxer openplenary-config.groovy al directory d'execució del servidor d'aplicacions. Si no saps quin és, mira als logs del servidor el missatge "Application running directory" per veure el path.

Les opcions de configuració són:

  • grails.serverURL: Necessari per generar correctament l'enllaç de confirmació de registre de l'usuari
  • grails.openplenary.defaultLocale: Idioma de l'aplicació, per defecte: new Locale("en","EN")
  • dataSource: Per defecte es fa servir una base de dades H2 a memòria. Mira la documentació de Grails per configurar la base de dades que necessitis
  • mail: Mira la documentació del plugin mail per les opcions de configuració
  • grails.openplenary.mail.from: Adreça de correu que es farà servir com a "from:"
  • grails.openplenary.layout: Plantilla/disseny amb el que es visualitzarà OpenPlenary. Per defecte "main"
  • grails.openplenary.style: Estil de color que s'aplicará a la plantilla (blue, brown, green, orange, red, violet or yellow). Per defecte "yellow"
  • grails.openplenary.name: Nom del poble, ciutat o entitat que fa servir OpenPlenary
  • grails.openplenary.voting.allowAnonymous: Permet vots anònims. Per defecte "false"
  • grails.openplenary.voting.maxAnonymousVotes: Màxim número de vots anònims permesos si la opció "grails.openplenary.voting.allowAnonymous" està activada. Per defecte 5
  • grails.openplenary.subject.slicePoint: Caracters on es tallaran els continguts dels punts. Per defecte 1000.
  • grails.openplenary.encodeEmail: Indica si es guardaran els emails codificats. Per defecte "false"
  • grails.openplenary.fileUploadPath: Directori del servidor on es pujaran els fitxers
  • grails.plugins.springsecurity.facebook.appId: Identificador de l'aplicació Facebook (necessari per login).
  • grails.plugins.springsecurity.facebook.secret: Text/contrassenya de l'aplicació Facebook (necessari per login).
